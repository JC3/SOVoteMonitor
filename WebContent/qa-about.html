About QA Viewer
This tool allows you to view the candidate questionnaire responses organized by question instead of by user. It works by looking for the questions in the QA post, then parsing each user's response by matching text in blockquotes with the question text. The back-end polls the response every 60 seconds, you view the cached content; edits and changes to the posts will be updated here within 60 seconds.
The styling is pretty ugly, and I'm not really a web developer. Alternate stylings or layout tips/suggestions are more than welcome.
If you spot any issues or have any suggestions, please post an issue on the GitHub issue page.
Please do not share outside of SO posts (Twitter, etc.), thanks!
